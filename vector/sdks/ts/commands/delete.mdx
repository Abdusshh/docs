---
title: Remove
---

Used to delete vectors with array of IDs

## Arguments

<ResponseField name="IDs" type="string[] | number[]" required>
  The IDs of the vectors you want to delete.
</ResponseField>
<ResponseField name="namespace" type="{ namespace?: string }">
  Namespace to delete from. If not set, default namespace is used.
</ResponseField>

## Response

<ResponseField name="delete" type="number" required>
  The number of the deleted vectors.
</ResponseField>

<RequestExample>

```typescript Basic
const responseDelete = await index.delete(["2", "3"]);
//{ deleted: 2}
```

```typescript Namespace
const responseDelete = await index.delete(["2", "3"], { namespace: "my-namespace" });
//{ deleted: 2}
```

</RequestExample>
