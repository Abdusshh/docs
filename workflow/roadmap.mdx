---
title: "Roadmap"
---

- Enhance the Upstash Workflow SDK to expose and configure all QStash features:
    - Ordered execution
    - Rate limit    
- Expand language support with additional SDKs: Python and other languages based on user requests.
- Local development environment for testing
- Endpoint for checking the status & progress of a workflow run
- Running in Vercel Preview: Currently it's not possible to run a workflow in Vercel Preview unless authentication
is disabled.
