---
title: GET
description: Return the value of the specified key or `None` if the key doesn't exist.
---


## Arguments 

<ParamField body="key" type="str" required>
  The key to get.
</ParamField>

## Response

<ResponseField  required>
The response is the value stored at the key or `None` if the key doesn't exist.
</ResponseField>



<RequestExample>
```py Example
redis.set("key", "value")

assert redis.get("key") == "value"
```
</RequestExample>
