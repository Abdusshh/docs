---
title: SISMEMBER
description: Check if a member exists in a set
---

## Arguments

<ParamField body="key" type="str" required>
  The key of the set to check.
</ParamField>

<ParamField body="member" type="TMember">
  The member to check for.
</ParamField>


## Response

<ResponseField type="0 | 1" required>
  `1` if the member exists in the set, `0` if not.
</ResponseField>

<RequestExample>
```py Example 
redis.sadd("set", "a", "b", "c"); 
isMember =  redis.sismember("set", "a")
print(isMember); # 1
```

</RequestExample>
