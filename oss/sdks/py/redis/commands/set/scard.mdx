---
title: SCARD
description: Return how many members are in a set
---

## Arguments

<ParamField body="key" type="str" required>
  The key of the set.
</ParamField>


## Response

<ResponseField type="int" required>
  How many members are in the set.
</ResponseField>

<RequestExample>
```py Example 
redis.sadd("key", "a", "b", "c"); 

assert redis.scard("key") == 3
```

</RequestExample>
