---
title: SRANDMEMBER
description: Returns one or more random members from a set.
---

## Arguments

<ParamField body="key" type="str" required>
  The key of the set.
</ParamField>

<ParamField body="count" type="number" default={1}>
  How many members to return.
</ParamField>


## Response

<ResponseField type="TMember | TMember[]" required>
  The random member.
  If `count` is specified, an array of members is returned.
</ResponseField>

<RequestExample>
```py Example 
redis.sadd("set", "a", "b", "c"); 
member = redis.srandmember("set")
print(member); # "a"
```
```py With Count 
redis.sadd("set", "a", "b", "c"); 
members = redis.srandmember("set", 2)
print(members); # ["a", "b"]
```

</RequestExample>
