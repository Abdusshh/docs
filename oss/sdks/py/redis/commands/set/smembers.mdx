---
title: SMEMBERS
description: Return all the members of a set
---

## Arguments

<ParamField body="key" type="str" required>
  The key of the set.
</ParamField>

## Response

<ResponseField type="TMember[]" required>
  The members of the set.
</ResponseField>

<RequestExample>
```py Example 
redis.sadd("set", "a", "b", "c"); 
members =  redis.smembers("set")
print(members); # ["a", "b", "c"]
```

</RequestExample>
