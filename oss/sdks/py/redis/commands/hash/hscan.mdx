---
title: HSCAN
description: Scan a hash for fields.
---

## Arguments

<ParamField body="key" type="string" required>
  The key of the hash.
</ParamField>

<ParamField body="cursor" type="number">
  The cursor, use `0` in the beginning and then use the returned cursor for subsequent calls.
</ParamField>

<ParamField body="options" type="Object">
  <ParamField body="match" type="string">
    Glob-style pattern to filter by field names.
  </ParamField>
  <ParamField body="count" type="number">
    Number of fields to return per call.
  </ParamField>
</ParamField>

## Response

<ResponseField type="[number, string[]]" required>
  The new cursor and the fields.
  If the new cursor is `0` the iteration is complete.
</ResponseField>

<RequestExample>
```py Basic
redis.hset("key", {
  id: 1,
  username: "chronark",
  name: "andreas"
 })
[newCursor, fields] = redis.hscan("key", 0)
print(newCursor); # likely `0` since this is a very small hash
print(fields); # ["id", "username", "name"]
```

```py Match
redis.hset("key", {
  id: 1,
  username: "chronark",
  name: "andreas",
})
[newCursor, fields] = redis.hscan("key", 0, { match: "user*" })
print(newCursor); # likely `0` since this is a very small hash
print(fields); # ["username"]
```


```py Count
redis.hset("key", {
  id: 1,
  username: "chronark",
  name: "andreas",
})
[newCursor, fields] = redis.hscan("key", 0, { count: 2 })
print(fields); # ["username", "id"]
```
</RequestExample>
