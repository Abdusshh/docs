---
title: HEXISTS
description: Checks if a field exists in a hash.
---


## Arguments

<ParamField body="key"type="str" required>
  The key to get.
</ParamField>

<ParamField body="field"type="str" required>
 The field to check.
</ParamField>

## Response

<ResponseField  type="integer" required>
   `1` if the hash contains `field`. `0` if the hash does not contain `field`, or `key` does not exist.
</ResponseField>

<RequestExample>
```py Example
redis.hset("key", "field", "value")
exists = redis.hexists("key", "field")
print(exists); # 1

```
</RequestExample>
