---
title: HKEYS
description: Return all field names in the hash stored at key.
---

## Arguments

<ParamField body="key" type="str" required>
  The key of the hash.
</ParamField>


## Response

<ResponseField type="string[]" required>
  The field names of the hash
</ResponseField>

<RequestExample>
```py Example
redis.hset("key", {
  id: 1,
  username: "chronark",
  })
fields = redis.hkeys("key")
print(fields); # ["id", "username"]
```
</RequestExample>
