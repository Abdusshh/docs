---
title: HSET
description: Write one or more fields to a hash.
---

## Arguments

<ParamField body="key" type="str" required>
  The key of the hash.
</ParamField>

<ParamField body="field" type="Optional[str]">
  Field to set
</ParamField>

<ParamField body="value" type="Optional[str]" required>
  Value to set
</ParamField>

<ParamField body="fields" type="Optional[Dict[str, Any]]">
  An object of fields and their values.
</ParamField>


## Response

<ResponseField type="int" required>
  The number of fields that were added.
</ResponseField>

<RequestExample>
```py Example
# Set a single field
assert redis.hset("myhash", "field1", "Hello") == 1

# Set multiple fields
assert redis.hset("myhash", values={
  "field1": "Hello",
  "field2": "World"
}) == 2
```
</RequestExample>
