---
title: HRANDFIELD
description: Return a random field from a hash
---

## Arguments

<ParamField body="key" type="string" required>
  The key of the hash.
</ParamField>

<ParamField body="count" type="integer">
  Optionally return more than one field.
</ParamField>

<ParamField body="withValues" type="boolean">
  Return the values of the fields as well.
</ParamField>

## Response

<ResponseField type="Record<string, unknown>" required>
  An object containing the fields and their values.
</ResponseField>

<RequestExample>
```py Basic
redis.hset("key", {
  id: 1,
  username: "chronark",
  name: "andreas"
 })
randomField = redis.hrandfield("key")
print(randomField); # one of "id", "username" or  "name"
```

```py Multiple Fields
redis.hset("key", {
  id: 1,
  username: "chronark",
  name: "andreas",
})
randomFields = redis.hrandfield("key", 2)
print(randomFields); # ["id", "username"] or any other combination
```

```py With Values
redis.hset("key", {
  id: 1,
  username: "chronark",
  name: "andreas",
})
randomFields = redis.hrandfield("key", 2, true)
print(randomFields); # { id: "1", username: "chronark" } or any other combination
```

</RequestExample>
