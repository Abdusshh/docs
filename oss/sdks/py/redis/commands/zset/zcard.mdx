---
title: ZCARD
description: Returns the number of elements in the sorted set stored at key.
---


## Arguments 

<ParamField body="key" type="str" required>
  The key to get.
</ParamField>

## Response

<ResponseField type="integer" required>
    The number of elements in the sorted set.
</ResponseField>



<RequestExample>
```py Example

redis.zadd("key", 
    { score: 1, member: "one"}, 
    { score: 2, member: "two" },
)
elements = redis.zrank("key")
print(elements); # 2
```
</RequestExample>
