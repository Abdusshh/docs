---
title: TYPE
description: Get the type of a key.
---


## Arguments

<ParamField body="key" type="string" required>
  The key to get.
</ParamField>

## Response

<ResponseField type="string" required>
  The type of the key.

  One of `string` | `list` | `set` | `zset` | `hash` | `none`
</ResponseField>

<RequestExample>
```py Example
await redis.set("key", "value")
t = await redis.type("key")
print(t) # "string"
```
</RequestExample>
