---
title: RENAMENX
description: Rename a key if it does not already exist.
---


## Arguments

<ParamField body="source" type="str" required>
  The original key.
</ParamField>


<ParamField body="destination" type="str" required>
  A new name for the key.
</ParamField>

## Response

<ResponseField type="0 | 1" required>
 `1` if key was renamed, `0` if key was not renamed.
</ResponseField>

<RequestExample>
```py Example
renamed = redis.rename("old", "new")
```
</RequestExample>
