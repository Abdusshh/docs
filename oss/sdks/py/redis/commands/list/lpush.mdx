---
title: LPUSH
description: Push an element at the head of the list.
---

## Arguments

<ParamField body="key" type="str" required>
  The key of the list.
</ParamField>

<ParamField body="elements" type="...TValue[]" required>
  One or more elements to push at the head of the list.
</ParamField>


## Response

<ResponseField type="number" required>
  The length of the list after the push operation.
</ResponseField>

<RequestExample>
```py Example 
length1 = redis.lpush("key", "a", "b", "c"); 
print(length1); # 3
length2 = redis.lpush("key", "d"); 
print(length2); # 4
 ```


</RequestExample>
