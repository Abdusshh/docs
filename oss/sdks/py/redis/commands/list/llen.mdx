---
title: LLEN
description: Returns the length of the list stored at key.
---



## Arguments

<ParamField body="key" type="str" required>
    The key of the list.
</ParamField>

## Response

<ResponseField type="int" required>
    The length of the list at key.
</ResponseField>

<RequestExample>
```py Example
redis.rpush("key", "a", "b", "c")

assert redis.llen("key") == 3
```
</RequestExample>
