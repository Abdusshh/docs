---
title: LPUSHX
description: Push an element at the head of the list only if the list exists.
---

## Arguments

<ParamField body="key" type="str" required>
  The key of the list.
</ParamField>

<ParamField body="elements" type="...TValue[]" required>
  One or more elements to push at the head of the list.
</ParamField>


## Response

<ResponseField type="number" required>
  The length of the list after the push operation.

  `0` if the list did not exist and thus no element was pushed.
</ResponseField>

<RequestExample>
```py Example 
redis.lpush("key", "a", "b", "c"); 
length = redis.lpushx("key", "d"); 
print(length); # 4
 ```

 ```py Without existing list 
length = redis.lpushx("key", "a"); 
print(length); # 0
 ```


</RequestExample>
