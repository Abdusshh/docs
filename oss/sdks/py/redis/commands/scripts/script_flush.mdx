---
title: SCRIPT FLUSH
description: Removes all scripts from the script cache.
---

## Arguments

<ParamField body="flush_type" type='"ASYNC" | "SYNC"' required>
  Whether to perform the flush asynchronously or synchronously.
</ParamField>

<RequestExample>
```py Example
redis.script_flush(flush_type="ASYNC")
```
</RequestExample>
