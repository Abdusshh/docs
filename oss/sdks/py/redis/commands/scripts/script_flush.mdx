---
title: SCRIPT FLUSH
description: Removes all scripts from the script cache.
---

## Arguments

<ParamField body="hahes" type="List[str]" required>
  The sha1 of the scripts to check.
</ParamField>

## Response

<ResponseField type="(0 | 1)[]" required>
  For every script that exists, an element of the array will be 1, otherwise 0.
</ResponseField>

<RequestExample>
```py Example
result = redis.scriptExists([sha])
print(result) # [0] or [1] depending on if the script exists or not.

```
</RequestExample>
