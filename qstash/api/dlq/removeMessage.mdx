---
title: "Remove a message from the DLQ"
description: "Remove a message to handle manually"
api: "DELETE /v2/dlq/messages/:messageId"
authMethod: "bearer"

---

Remove a message from the DLQ, it is now up to you to handle it in any way you want.

## Request

<ParamField path="messageId" type="string" required >
The id of the message you want to remove from the DLQ.
</ParamField>



## Response

This endpoint returns `200 OK` or `404`, depending on whether the message existed in the DLQ or not.



<RequestExample>



```sh 
curl https://qstash.upstash.io/v2/dlq/messages \
  -H "Authorization: Bearer <token>"
```




</RequestExample>

<ResponseExample>
```text 200 OK
OK
```
```text 404 NOT FOUND
```

</ResponseExample>
